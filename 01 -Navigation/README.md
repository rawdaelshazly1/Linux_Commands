# 01 - Navigation

## Objective

In this chapter, I learned the basic Linux navigation commands and practiced using them in the terminal.

## Commands I Learned

- pwd – Show the current working directory.
- ls – List files and directories.
- ls -l – Show detailed information.
- ls -la – Show hidden files.
- cd – Change directory.
- mkdir – Create a new directory.
- touch – Create a new file.

## Practice

I created a simple folder structure using Linux commands:

## What I Learned

- How to move between directories.
- The difference between absolute and relative paths.
- How to create folders and files.
- How to display hidden files.

## Notes

I noticed that ls -la is useful because it shows hidden files, which can be important when working on Linux systems.
